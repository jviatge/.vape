Vape is the simplest and fastest way to build production-ready administration panels using nextjs

## Install

1 - clone vape

```BASH
git clone https://github.com/jviatge/vape.git && cd vape

```

2 - Run init

```BASH
pnpm init-project

```

3 - Install

```BASH
pnpm install

```

4 - Generate database

```BASH
pnpm prisma:generate
```
